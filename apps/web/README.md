# Web App Foundation

Static browser-first scaffold for Cloudflare Pages deployment.

## Local Preview

Open `apps/web/index.html` directly in a browser, or run any static file server.

## Runtime Config

- Default config: `apps/web/config/default.json`
- Deploy-time config: `apps/web/config/runtime.json` (generated by workflow)
- URL overrides for validation: `?apiBaseUrl=...&artifactBaseUrl=...&environment=...`
  (enabled only for local contexts or when `enableQueryOverrides=true` in config)

## Smoke Test

```bash
python scripts/web/smoke_test.py --base-dir apps/web
```
